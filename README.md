# LANGCHAIN DEMO

Proyecto práctico y educativo para aprender a interactuar con nuestros datos usando Langchain.

Para ejecutar la api de flask:

    python3 -m venv venv

    source venv/bin/activate

    pip install -r requirements.txt

    python3 {FILENAME}.py
